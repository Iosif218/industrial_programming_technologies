# Описание сущностей
## Customer (покупатель)
Содержит информацию о покупателе
## Order (заказ)
Содержит информацию о заказе
## Order/Product
Обеспечивает свзяь многое ко многим между заказом и товарах
## Product (товар)
Сожержит информацию о товарах
